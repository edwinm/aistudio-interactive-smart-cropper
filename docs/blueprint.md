# **App Name**: Smart Crop

## Core Features:

- Image Upload: Drag and drop images into the application.
- Dimension Input: Set custom dimensions (width and height) with default values of 600x400 pixels.
- Smart Crop: AI powered tool that intelligently crops the image, ensuring important parts are within the cropped area.
- Cropped Image Preview: Provide users with preview functionality so they can preview before downloading the file
- Image Download: Download the cropped image to local storage in webp format.

## Style Guidelines:

- Primary color: Forest green (#4CAF50) to evoke a sense of natural imagery and precision.
- Background color: Light green (#F1F8E9), providing a subtle and clean backdrop.
- Accent color: Yellow-Green (#8BC34A) for interactive elements and highlights.
- Clean, sans-serif font for readability and a modern feel.
- Use simple, intuitive icons for actions like upload, crop, and download.
- A clean, centered layout to focus user attention on the image cropping process.
